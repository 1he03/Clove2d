## Overview

v0.7.0 adds SVG import and export capabilities.

### Key Features

1. **SVG Import**
   - Parse SVG files
   - Convert to Clove2d objects
   - Handle transformations
   - Support common SVG features

2. **SVG Export**
   - Export canvas as SVG
   - Preserve layers
   - Export with metadata
   - Optimize output

3. **Vector Operations**
   - Maintain vector quality
   - Resolution-independent scaling
   - Better quality at any size
