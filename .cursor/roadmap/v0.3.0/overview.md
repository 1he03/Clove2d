## Overview

v0.3.0 expands shape capabilities and adds advanced image filters.

### Key Features

1. **Advanced Shapes**
   - Complex bezier curves
   - Custom path builder
   - Boolean operations (union, intersect, subtract)
   - Shape morphing

2. **Advanced Filters**
   - Edge detection
   - Emboss
   - Oil painting effect
   - Pixelate
   - Mosaic
   - Custom convolution kernels
   - Filter masks
