## Overview

v0.8.0 adds 3D transformation support for advanced visual effects.

### Key Features

1. **3D Transforms**
   - 3D rotation (X, Y, Z axes)
   - Perspective transforms
   - 3D translation
   - Transform origin

2. **Projection**
   - Perspective projection
   - Orthographic projection
   - Depth sorting
