## Overview

v0.5.0 adds frame-based animation capabilities.

### Key Features

1. **Animation System**
   - Timeline management
   - Keyframe animation
   - Easing functions
   - Animation curves

2. **Frame Export**
   - Frame-by-frame rendering
   - GIF export
   - Video export (optional)
   - Sprite sheet generation
